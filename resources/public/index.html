Sorsa Source Storage API

Sorsa is used through an HTTP API. The following requests are supported:

GET /:folder/list?password=:password
Returns contents of a folder
GET /:folder/document/:document?password=:password
Returns contents of a document
POST /:folder/document/:document?content=:content&password=:password
Inserts a document with content :content to the storage. If the folder does not exist, sets the password.
